# 🧬 AI-Based Health Risk Prediction System  
### Novagen Research Lab  

An end-to-end Machine Learning web application that predicts whether an individual is **Healthy** or **Unhealthy** based on lifestyle and physical health parameters.

Built using a **Random Forest Classifier (93% accuracy)** and deployed with **Streamlit** for an interactive user experience.

---

## 🚀 Project Overview

This system analyzes multiple health indicators such as:

- Age
- BMI
- Blood Pressure
- Cholesterol
- Glucose Level
- Sleep Hours
- Exercise Hours
- Stress Level
- Lifestyle habits (Smoking, Alcohol)
- Diet Type
- Blood Group
- Medical History
- And more...

The model predicts the health risk category and provides:
- ✅ Direct classification output (Healthy / Unhealthy)
- 📊 Confidence probability scores
- 📈 Feature importance visualization
- 🧠 Clinical insight interpretation

---

## 🏗️ Tech Stack

- Python
- Scikit-learn
- Random Forest Classifier
- Pandas / NumPy
- Plotly (Data Visualization)
- Streamlit (Deployment Interface)
- Joblib (Model Serialization)

---

## 📊 Model Performance

- Algorithm: Random Forest Classifier
- Accuracy: **93%**
- No scaling applied to target variable
- Feature alignment handled dynamically using `model.feature_names_in_`
